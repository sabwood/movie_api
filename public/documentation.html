Documentation: Movie API

A server-side component of a "movies" web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
Get a list of all movies /movies GET None A JSON object holding data about all movies.
Get data about a single movie by title /movies/[title] GET None A JSON object holding data about a single movie, containing the title, release year, and director.
Get data about a genre by movie title /movies/genre/[genreName] GET None A JSON object holding data about a single genre, containing a genre name and description.
Get data about a director by name /movies/directors/[directorName] GET None A JSON object holding data about a single director, containing director name, biography, birth year, and death year.
Add new user /users POST A JSON object holding data about the user to add, structured like: { 'username': 'exampleUsername', 'Name': 'exampleName', 'favoriteMovies': [] } A JSON object holding data about the user that was added, including an ID. Example: { 'id': '#####', 'username': 'exampleUsername', 'Name': 'exampleName', 'favoriteMovies': [] }
Allow users to update their username /users/[id] PUT A JSON object holding data about the user which needs to be updated. Example: { 'id': '#####', 'username': 'newUsername', 'name': 'exampleName', 'favoriteMovies': [] } A JSON object holding data about the updated user information. Example: { 'id': '#####', 'username': 'newUsername', 'name': 'exampleName', 'favoriteMovies': [] }
Allow users to add a movie to their favorites list /users/[id]/[movieTitle] POST None A text message indicating whether the user added the movie to their favorites list successfully.
Allow users to remove a movie from their favorites list /users/[id]/[movieTitle] DELETE None A text message indicating whether the user removed the movie from their favorites list successfully.
Allow existing users to deregister /users/[id] DELETE None A text message indicating whether the user deregistered unsuccessfully.
